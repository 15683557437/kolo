# 必应日图

### 接口说明

直接返回必应搜索每天用的壁纸

### 接口文档

接口链接:`http://api.2kolo.cn/api/bing/api.php`

返回格式:`IMG`  请求方式:`GET`

### 请求参数

|  名称  |  类型  |  必填  |  说明  |
|  :----:  |  :----:  |  :----:  |  :----:  |
|  无  |  null  |  否  |无需参数|

### 返回参数

|名称|类型|说明
|:----:|:----:|:----:|
|img|null|无参数，直接返回图片|

### 请求示例
