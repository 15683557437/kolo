# 随机头像

### 接口说明

随机返回各种类型的头像图片

### 接口文档

接口链接:`http://api.2kolo.cn/api/sjtx/api.php`

返回格式:`IMG`  请求方式:`GET`

### 请求参数

|  名称  |  类型  |  必填  |  说明  |
|  :----:  |  :----:  |  :----:  |  :----:  |
|  lx  |  String  |  是  |  a1(男头)   b1(女头)   c1(动漫)   c2(动漫男头)   c3(动漫女头)|

### 返回参数

|名称|类型|说明
|:----:|:----:|:----:|
|img|null|无参数，直接返回图片|

### 请求示例

`1:`http://api.huokuoluo.cn/api/sjtx/api.php?lx=c1

`2:`http://api.huokuoluo.cn/api/sjtx/api.php?lx=b1
